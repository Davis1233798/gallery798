一個專為手機優化的遊戲入口網站，提供 500+ 個免費 HTML5 遊戲，無需下載即可暢玩！

- 🎯 **500+ 遊戲** - 豐富的遊戲庫，涵蓋各種類型

## 🛠️ 安裝和使用

### 直接使用
1. 訪問網站：[遊戲天堂](https://davis1233798.github.io/game_portal/)
2. 選擇您喜歡的遊戲類型
3. 點擊遊戲卡片開始遊玩
4. 享受遊戲時光！

### PWA 安裝
1. 在支援的瀏覽器中打開網站
2. 點擊地址欄的"安裝"圖標
3. 確認安裝到主畫面
4. 從主畫面啟動應用

### 本地開發
```bash
# 克隆項目
git clone https://github.com/Davis1233798/game_portal.git

# 進入項目目錄
cd game_portal

# 使用本地服務器運行
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx serve .

# 使用 PHP
php -S localhost:8000

# 訪問 http://localhost:8000
```

## 🎯 快捷鍵
